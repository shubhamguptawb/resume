## How To Use

###### To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [yarn](https://yarnpkg.com) installed on your computer).

1. Fork this repository and clone the project

###### Please give me proper credit by linking back to [shubham-gupta.vercel.app](https://shubham-gupta.vercel.app/).

```bash
  git clone https://github.com/shubham-gupta/portfolio-website.git
```

2. Go to the project directory

```bash
  cd portfolio-website
```

3. Install dependencies

```bash
  yarn
```

4. Start the server

```bash
  yarn dev
```

- To change the content, edit `src/utils/portfolio.ts`

- To change the **resume**, change `/public/resume.pdf` with your resume (name must be `resume.pdf`)

- To change the **About Section**, edit `src/containers/About.tsx`

- To change the theme, edit `src/styles/globals.css`
